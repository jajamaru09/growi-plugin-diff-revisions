# 01-requirements.md - 初回実装の要求内容

## 変更・追加する機能の説明

GROWIプラグイン「growi-plugin-diff-revisions」の全機能を初回実装する。

1. **プラグイン基盤**: GROWIプラグイン仕様（スキーマv4、scriptタイプ）に準拠した基盤構築
2. **サイドバーボタン**: 差分比較モーダルを開くボタンをサイドバーに配置
3. **差分比較モーダル**: 左右分割レイアウトのモーダルUI
4. **リビジョン取得**: GROWI APIからリビジョン一覧を取得（offsetループ対応）
5. **リビジョン選択**: ドロップダウンで「#No - revisionId - YYYY/MM/DD HH:mm 更新者」形式で選択
6. **Markdownレンダリング**: unified/remark/rehypeによるMarkdown→HTML変換
7. **CSS適用**: `.wiki` クラスによるGROWI CSSの継承
8. **差分検出・ハイライト**: htmldiff-jsによるHTML差分検出と色分けハイライト

## ユーザーストーリー

- 記事閲覧中にサイドバーのボタンをクリックし、モーダルで2つのリビジョンを選択して差分を確認できる
- ドロップダウンでリビジョン番号・ID・日時・更新者を確認して適切なリビジョンを選べる

## 受け入れ条件

- [ ] `npm run build` でビルドが成功する
- [ ] GROWIにプラグインとして読み込まれ、サイドバーにボタンが表示される
- [ ] ボタンクリックでモーダルが開き、左右パネルが表示される
- [ ] リビジョン一覧がドロップダウンに正しい形式で表示される
- [ ] リビジョン選択でHTMLレンダリングされた内容が表示される
- [ ] 2つのリビジョン選択時に差分がピンク/緑でハイライトされる
- [ ] モーダルを閉じることができる（Escキー、×ボタン、背景クリック）

## 制約事項

- GROWIの内部モジュール（growiFacade等）は利用不可
- GROWI固有のディレクティブ（lsx等）のレンダリングは対応外
